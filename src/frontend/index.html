Yakusoku | Realtime appointments system

Get the code at http://github.com/yrezgui/yakusoku

What's that ?

Yakusoku is a simple open source project to show how simple it is to have a realtime appointments system. It run on NodeJS and uses different components :

  • Orchestrate for storing the data
  • Pusher for sending in realtime the new appointments across all the browsers
  • CLNDR.js for generating the calendar (I copy the UI of their website too)

How to get started

  1. Clone the repository http://github.com/yrezgui/yakusoku
  2. Execute this command from the cloned repository folder: npm install
  3. Create an account Orchestrate
  4. Create an account Pusher
  5. Set these environment variables:
    • PUSHER_APPID
    • PUSHER_KEY
    • PUSHER_SECRET
    • ORCHESTRATE_TOKEN
    • PORT (port of the NodeJS server)
  6. And finally: node web.js
